# 开发技术规范

## 核心原则

本项目采用统一的技术栈和开发规范，所有开发必须严格遵循以下规范。

**核心原则**：
- **RPC 是主要方式**：使用 Hono 的 RPC 特性调用 API
- **类型安全是核心**：使用 TypeScript 自动推导请求和响应类型
- **自定义 Hook 是推荐**：在自定义 Hook 中封装 RPC 和 React Query
- **错误处理是必须**：统一的错误处理函数

---

## 技术栈

| 技术 | 版本 | 作用 |
|------|------|------|
| **Next.js** | 14.2.5 | React 框架，支持 SSR、App Router、Server Components |
| **Hono** | 4.6.2 | 后端 API 框架，轻量、高性能 |
| **React Query** | 5.56.2 | 数据管理库，提供缓存、加载状态、乐观更新 |
| **TypeScript** | 5 | 类型安全，减少运行时错误 |
| **shadcn/ui** | - | 基于 Radix UI 的现代化组件库 |

## 核心概念

### 1. RPC（Hono Client）

**文件位置**：`lib/api-client.ts`

**作用**：像调用本地函数一样调用远程 API，提供类型安全和自动补全。

**优势**：
- 类型安全：TypeScript 自动检查参数和返回值
- 自动补全：IDE 提示所有可用的 API
- 无需记忆 URL：像调用函数一样调用 API

**使用方法**：

```typescript
import { client } from '@/lib/api-client'

// 获取数据
const res = await client.api.tasks.$get()
const data = await res.json()

// 创建数据
const res = await client.api.tasks.$post({ json: { title } })
const data = await res.json()

// 更新数据
const res = await client.api.tasks[':id'].$put({
  param: { id },
  json: { completed: true },
})
const data = await res.json()

// 删除数据
const res = await client.api.tasks[':id'].$delete({
  param: { id },
})
const data = await res.json()
```

### 2. 类型推导

**作用**：使用 `InferRequestType` 和 `InferResponseType` 自动推导请求和响应类型。

**示例**：

```typescript
import { client } from '@/lib/api-client'
import { InferRequestType, InferResponseType } from 'hono/client'

const $post = client.api.tasks.$post
type PostRequestType = InferRequestType<typeof $post>['json']
type PostResponseType = InferResponseType<typeof $post>['data']

// PostRequestType = { title: string }
// PostResponseType = { data: Task }
```

### 3. 自定义 Hook

**作用**：封装 RPC 和 React Query，提供类型安全、代码复用、统一处理。

**优势**：
- 类型安全：自动推导请求和响应类型
- 代码复用：避免重复编写相同的逻辑
- 统一处理：错误处理、加载状态、成功提示都在 Hook 中

- `server/api/`：API 路由

### 2. 错误处理

#### ✅ 必须使用：统一的错误处理

**文件位置**：`server/api/error.ts`

**错误处理函数**：

```typescript
import { z } from 'zod'
import type { Context } from 'hono'
import { HTTPException } from 'hono/http-exception'
import type { StatusCode } from 'hono/utils/http-status'

export class ApiError extends HTTPException {
  public readonly code?: StatusCode

  constructor({ code, message }: { code?: StatusCode; message: string }) {
    super(code, { message })
    this.code = code
  }
}

export function handleError(err: Error, c: Context): Response {
  if (err instanceof z.ZodError) {
    const firstError = err.issues[0]

    return c.json(
      { code: 422, message: `\`${firstError.path}\`: ${firstError.message}` },
      422,
    )
  }

  return c.json(
    {
      code: 500,
      message: '出了点问题, 请稍后再试。',
    },
    { status: 500 },
  )
}
```

**在 API 入口文件中绑定错误处理**：

```typescript
import { handleError } from './error'

const app = new Hono().basePath('/api')
app.onError(handleError)
```

**优势**：
- 统一的错误响应格式
- 自动处理 Zod 验证错误
- 自动处理业务错误

### 3. 自定义 Hook 规范

#### ✅ 推荐使用：类型安全的自定义 Hook

**文件位置**：`hooks/use-tasks.ts`

**使用方法**：

```typescript
import { client } from '@/lib/api-client'
import { InferRequestType, InferResponseType } from 'hono/client'
import { useMutation, useQuery } from '@tanstack/react-query'
import { toast } from 'sonner'

const $get = client.api.tasks.$get
type GetResponseType = InferResponseType<typeof $get>['data']

const $post = client.api.tasks.$post
type PostRequestType = InferRequestType<typeof $post>['json']
type PostResponseType = InferResponseType<typeof $post>['data']

const $put = client.api.tasks[':id'].$put
type PutRequestType = InferRequestType<typeof $put>['json']
type PutResponseType = InferResponseType<typeof $put>['data']

const $delete = client.api.tasks[':id'].$delete
type DeleteResponseType = InferResponseType<typeof $delete>['data']

export function useTasks() {
  return useQuery<GetResponseType, Error>({
    queryKey: ['tasks'],
    queryFn: async () => {
      const res = await $get()
      return (await res.json()).data
    },
  })
}

export function useCreateTask() {
  return useMutation<PostResponseType, Error, PostRequestType>({
    mutationFn: async (json) => {
      const res = await $post({ json })
      return (await res.json()).data
    },
    onSuccess: () => {
      toast.success('任务创建成功')
    },
    onError: (error) => {
      toast.error('创建失败：' + error.message)
    },
  })
}

export function useUpdateTask() {
  return useMutation<PutResponseType, Error, PutRequestType>({
    mutationFn: async ({ id, ...json }) => {
      const res = await $put({ param: { id }, json })
      return (await res.json()).data
    },
    onSuccess: () => {
      toast.success('任务更新成功')
    },
    onError: (error) => {
      toast.error('更新失败：' + error.message)
    },
  })
}

export function useDeleteTask() {
  return useMutation<DeleteResponseType, Error, string>({
    mutationFn: async (id) => {
      const res = await $delete({ param: { id } })
      return (await res.json()).data
    },
    onSuccess: () => {
      toast.success('任务删除成功')
    },
    onError: (error) => {
      toast.error('删除失败：' + error.message)
    },
  })
}
```

**在组件中使用**：

```typescript
'use client'

import { useTasks, useCreateTask, useUpdateTask, useDeleteTask } from '@/hooks/use-tasks'
import { useQueryClient } from '@tanstack/react-query'

export default function TasksPage() {
  const queryClient = useQueryClient()
  const { data, isLoading } = useTasks()
  const createTask = useCreateTask()
  const updateTask = useUpdateTask()
  const deleteTask = useDeleteTask()

  const handleSubmit = (e: React.FormEvent) => {
    e.preventDefault()
    createTask.mutate({ title: '新任务' })
  }

  return (
    <div>
      <form onSubmit={handleSubmit}>
        <button type="submit" disabled={createTask.isPending}>
          添加
        </button>
      </form>

      <div>
        {data?.data?.map((task) => (
          <div key={task.id}>
            <span>{task.title}</span>
            <button onClick={() => updateTask.mutate({ id: task.id, completed: !task.completed })}>
              切换状态
            </button>
            <button onClick={() => deleteTask.mutate(task.id)}>
              删除
            </button>
          </div>
        ))}
      </div>
    </div>
  )
}
```

**优势**：
- 类型安全：自动推导请求和响应类型
- 代码复用：避免重复编写相同的逻辑
- 统一处理：错误处理、加载状态、成功提示都在 Hook 中

### 4. queryKey 命名规范

**规则**：
- 必须使用数组格式
- 第一个元素：资源名（复数）
- 后续元素：参数（可选）

**示例**：

```typescript
// ✅ 正确
queryKey: ['tasks']
queryKey: ['tasks', filter]
queryKey: ['task', id]

// ❌ 错误
queryKey: 'tasks'

```

### 5. React Query 使用规范

#### useQuery 使用规范

**何时使用**：获取数据（GET 请求）

**必须参数**：
- `queryKey`：查询的唯一标识（数组格式）
- `queryFn`：获取数据的异步函数

**必须处理的状态**：
- `isLoading`：加载状态
- `error`：错误

**示例**：

```typescript
const { data, isLoading, error } = useQuery({
  queryKey: ['tasks'],  // 必须：使用数组
  queryFn: async () => {  // 必须：异步函数
    const res = await client.api.tasks.$get()  // 必须：使用 RPC
    return res.json()
  },
})

// 必须处理加载状态
if (isLoading) {
  return <div>加载中...</div>
}

// 必须处理错误
if (error) {
  return <div>加载失败：{error.message}</div>
}

return <div>{data?.data?.map(...)}</div>
```

---

#### useMutation 使用规范

**何时使用**：修改数据（POST/PUT/DELETE 请求）

**必须参数**：
- `mutationFn`：执行变更的异步函数

**必须处理的回调**：
- `onSuccess`：成功回调（必须刷新相关查询）
- `onError`：错误回调（必须显示错误提示）

**必须使用的 Hook**：
- `useQueryClient`：用于刷新查询

**示例**：

```typescript
const queryClient = useQueryClient()

const createTask = useMutation({
  mutationFn: async (title: string) => {  // 必须：异步函数
    const res = await client.api.tasks.$post({ json: { title } })  // 必须：使用 RPC
    return res.json()
  },
  onSuccess: () => {  // 必须：刷新相关查询
    queryClient.invalidateQueries({ queryKey: ['tasks'] })
    toast.success('任务创建成功')
  },
  onError: (error) => {  // 必须：显示错误提示
    toast.error('创建失败：' + error.message)
  },
})

// 必须处理加载状态
const handleSubmit = (e: React.FormEvent) => {
  e.preventDefault()
  if (!newTaskTitle.trim()) return
  createTask.mutate(newTaskTitle)
}

return (
  <form onSubmit={handleSubmit}>
    <Input
      value={newTaskTitle}
      onChange={(e) => setNewTaskTitle(e.target.value)}
      disabled={createTask.isPending}  // 必须：禁用输入
    />
    <Button
      type="submit"
      disabled={createTask.isPending || !newTaskTitle.trim()}  // 必须：禁用按钮
    >
      添加
    </Button>
  </form>
)
```

**组件使用**：

```typescript
// app/dashboard/tasks/page.tsx
'use client'

import { useState } from 'react'
import { useTasks, useCreateTask, useUpdateTask, useDeleteTask } from '@/hooks/use-tasks'
import { useQueryClient } from '@tanstack/react-query'
import { toast } from 'sonner'
import { Button } from '@/components/ui/button'
import { Input } from '@/components/ui/input'
import { Checkbox } from '@/components/ui/checkbox'
import { Plus, Trash2 } from 'lucide-react'

export default function TasksPage() {
  const queryClient = useQueryClient()
  const [newTaskTitle, setNewTaskTitle] = useState('')
  const [filter, setFilter] = useState<'all' | 'pending' | 'completed'>('all')

  const { data, isLoading } = useTasks()
  const createTask = useCreateTask()
  const updateTask = useUpdateTask()
  const deleteTask = useDeleteTask()

  const handleSubmit = (e: React.FormEvent) => {
    e.preventDefault()
    if (!newTaskTitle.trim()) return
    createTask.mutate(newTaskTitle)
    setNewTaskTitle('')
  }

  const filteredTasks = data?.data?.filter((task) => {
    if (filter === 'pending') return !task.completed
    if (filter === 'completed') return task.completed
    return true
  })

  const completedCount = data?.data?.filter(t => t.completed).length || 0
  const totalCount = data?.data?.length || 0

  if (isLoading) {
    return (
      <div className="min-h-screen py-20">
        <div className="max-w-2xl mx-auto px-6">
          <p className="text-muted-foreground text-sm">加载中...</p>
        </div>
      </div>
    )
  }

  return (
    <div className="min-h-screen py-20">
      <div className="max-w-2xl mx-auto px-6">
        <div className="space-y-16">
          <div>
            <h1 className="text-5xl md:text-6xl font-semibold tracking-tight mb-4 text-primary">
              任务管理
            </h1>
            <p className="text-xl text-muted-foreground">
              {completedCount} / {totalCount} 已完成
            </p>
          </div>

          <form onSubmit={handleSubmit} className="flex gap-3">
            <Input
              type="text"
              placeholder="添加新任务..."
              value={newTaskTitle}
              onChange={(e) => setNewTaskTitle(e.target.value)}
              disabled={createTask.isPending}
              className="flex-1"
            />
            <Button
              type="submit"
              disabled={createTask.isPending || !newTaskTitle.trim()}
            >
              <Plus className="h-4 w-4 mr-2" />
              添加
            </Button>
          </form>

          <div className="space-y-4">
            <div className="flex items-center gap-2">
              {(['all', 'pending', 'completed'] as const).map((f) => (
                <Button
                  key={f}
                  variant={filter === f ? 'default' : 'ghost'}
                  size="sm"
                  onClick={() => setFilter(f)}
                  className="h-8"
                >
                  {f === 'all' ? '全部' : f === 'pending' ? '待处理' : '已完成'}
                  {f === 'all' && totalCount > 0 && ` (${totalCount})`}
                  {f === 'pending' && totalCount - completedCount > 0 && ` (${totalCount - completedCount})`}
                  {f === 'completed' && completedCount > 0 && ` (${completedCount})`}
                </Button>
              ))}
            </div>

            <div className="border-t pt-4">
              {filteredTasks && filteredTasks.length > 0 ? (
                <div className="space-y-3">
                  {filteredTasks.map((task) => (
                    <div
                      key={task.id}
                      className="group flex items-center gap-4 py-2"
                    >
                      <Checkbox
                        checked={task.completed}
                        onCheckedChange={() => updateTask.mutate({ id: task.id, completed: !task.completed })}
                      />
                      <span
                        className={`flex-1 text-base ${
                          task.completed
                            ? 'line-through text-muted-foreground'
                            : ''
                        }`}
                      >
                        {task.title}
                      </span>
                      <Button
                        variant="ghost"
                        size="icon"
                        className="h-8 w-8 opacity-0 group-hover:opacity-100 transition-opacity"
                        onClick={() => deleteTask.mutate(task.id)}
                        disabled={deleteTask.isPending}
                      >
                        <Trash2 className="h-4 w-4 text-muted-foreground hover:text-destructive" />
                      </Button>
                    </div>
                  ))}
                </div>
              ) : (
                <div className="py-12 text-center">
                  <p className="text-muted-foreground text-base">
                    {filter === 'all' ? '暂无任务，添加你的第一个任务吧' : `没有${filter === 'completed' ? '已完成' : '待处理'}的任务`}
                  </p>
                </div>
              )}
            </div>
          </div>
        </div>
      </div>
    </div>
  )
}
```

```typescript
'use client'

import { useState, useEffect } from 'react'
import axios from 'axios'

export default function TasksPage() {
  const [tasks, setTasks] = useState([])
  const [isLoading, setIsLoading] = useState(true)

  useEffect(() => {
    axios.get('/api/tasks').then(res => setTasks(res.data.data))
    setIsLoading(false)
  }, [])

  const createTask = async (title: string) => {
    await axios.post('/api/tasks', { title })
    const res = await axios.get('/api/tasks')
    setTasks(res.data.data)
  }

  return (
    <div>
      <input onChange={(e) => setNewTaskTitle(e.target.value)} />
      <button onClick={() => createTask(newTaskTitle)}>添加</button>

      <div>
        {tasks.map(task => (
          <div key={task.id}>{task.title}</div>
        ))}
      </div>
    </div>
  )
}
```

## 总结

### ✅ 必须使用

- **RPC（Hono Client）**：调用 API
- **类型推导（InferRequestType、InferResponseType）**：获取请求和响应类型
- **自定义 Hook**：封装 RPC 和 React Query
- **React Query**：管理数据
- **useQuery**：获取数据
- **useMutation**：修改数据
- **useQueryClient**：手动操作（仅在需要时）
- **shadcn/ui**：UI 组件
- **Tailwind CSS**：样式
- **统一的错误处理**：在 server/api/error.ts 中定义

### ❌ 禁止使用

- **fetch、axios、ky**：直接调用 API
- **useState + useEffect**：管理数据
- **SWR、Redux Toolkit Query、Apollo Client**：其他数据管理库
- **Material-UI、Ant Design、Chakra UI**：其他 UI 库
- **CSS Modules、Styled Components、CSS-in-JS**：其他样式方案

### ✅ 必须遵循

- 使用 RPC 调用 API
- 使用类型推导获取请求和响应类型
- 在自定义 Hook 中封装 RPC 和 React Query
- 在组件中使用自定义 Hook
- 处理 useQuery 的加载状态（isLoading）
- 处理 useMutation 的加载状态（isPending）
- 处理 useQuery 的错误（error）
- 处理 useMutation 的错误（onError）
- 刷新相关查询（invalidateQueries）
- 使用 shadcn/ui 组件
- 使用 Tailwind CSS 样式
- 在 server/api/error.ts 中定义统一的错误处理函数
- 在 server/api/index.ts 中绑定错误处理
