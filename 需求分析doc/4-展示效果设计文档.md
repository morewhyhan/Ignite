# 展示效果设计文档

---

## 一、核心设计理念

**用户打开系统的第一感受：好有道理，好爽，感觉用了这个东西马上能变成学霸，变成人上人，马上要暴富。**

这不是一个温和的学习工具，而是一个能让用户感觉**要起飞**的武器。

**核心驱动力：超强社会角色感觉 + 多巴胺驱动**

---

## 二、卡片等级系统设计

### 等级划分

| 等级 | 颜色 | 流光特效 | 卡片标准 |
|------|------|----------|----------|
| **N** | 灰色 | 无 | 基础：有定义，但模糊、不够准确 |
| **R** | 蓝色 | 微弱边框光晕 | 及格：定义清晰，但缺乏举例和关联 |
| **SR** | 紫色 | 边框流光 + 轻微粒子 | 良好：定义+举例，结构清晰 |
| **SSR** | 金色 | 边框流光 + 粒子特效 + 3D立体感 | 优秀：定义+举例+关联，逻辑严密 |
| **UR** | 彩虹渐变 | 全方位流光 + 粒子特效 + 3D立体感 + 背景图案 | 传说：定义+举例+关联+应用，体现深度理解与跨领域整合 |

### 卡片视觉设计

**所有等级卡片共有的特性：**
- Apple风格简约设计
- 标准结构：定义 + 举例 + 关联 + 应用
- 3D立体感，可以旋转查看
- 拖拽交互

**高等级卡片额外特性（SR及以上）：**
- 边框流光特效，颜色与等级对应
- 粒子特效，卡片表面闪烁
- 3D感更强，阴影更丰富

**UR卡片专属特性：**
- 背景有精美图案（根据领域定制）
- 流光特效更加炫酷
- 获得UR卡片时，全屏有特殊庆祝动画

## 三、插画背景场景设计

**风格参考：Shopify Editions Winter2026 —— 复古气质 + 3D交互 + 滚动长卷动画**

### 场景1：迷茫的学生（Hero区）

**视觉描述：**
- 一个年轻人坐在书桌前，周围是堆积如山的书和资料
- 眼神迷茫，不知道从哪里开始
- 背景是灰暗的，象征学习的沉重感

**滚动交互：**
- 随着用户向下滚动，背景逐渐明亮
- 书山开始分解，变成飘浮的知识碎片

### 场景2：选择领域

**视觉描述：**
- 不同领域的图标悬浮在空中（计算机、经济、文学、物理等）
- 每个领域对应一位巨匠的半身像（冯·诺依曼、凯恩斯、鲁迅、爱因斯坦等）
- 巨匠像有复古插画质感，但融入3D元素

**滚动交互：**
- 用户点击某个领域，背景切换成对应巨匠
- 巨匠的眼睛会跟随鼠标移动，仿佛在注视用户

### 场景3：巨匠对话

**视觉描述：**
- 左侧是巨匠的肖像（复古插画风格）
- 右侧是对话界面，显示聊天记录
- 背景是对话场景：一个古老的图书馆，巨匠与用户面对面坐着

**滚动交互：**
- 随着对话进行，图书馆逐渐明亮
- 巨匠的表情随着对话内容变化（思考、微笑、赞许）

### 场景4：卡片诞生

**视觉描述：**
- 用户说清概念后，Agent夸奖的文字出现
- bulingbuling音效播放
- 一张卡片从对话界面中"诞生"，飞到屏幕中央
- 屏幕右侧出现一个发光的卡片盒子

**交互流程：**
1. 用户说清概念 → Agent夸奖 → 音效 → 卡片出现
2. 卡片悬浮在屏幕中央，用户可以浏览、修改
3. 用户可以手动拖拽卡片进入盒子
4. 或者点击"归档"按钮，卡片变成流光，咻一下钻进盒子

**卡片诞生特效：**
- 卡片从对话界面中弹出，有光效
- 根据等级不同，特效不同（UR最炫酷）
- 拖拽时有物理反馈，卡片有重量感

### 场景5：知识宇宙

**视觉描述：**
- 一个巨大的知识图谱，概念像星星一样连接成星座
- 从稀疏到密集的动画，展示用户的成长
- 背景是深蓝色的宇宙，星星闪烁

**滚动交互：**
- 随着用户滚动，知识图谱逐渐变密集
- 用户可以点击任意概念，查看详情
- 高亮显示用户刚刚创建的卡片

### 场景6：蜕变升华

**视觉描述：**
- 左侧：迷茫的学生（场景1）
- 右侧：自信的领域领军者
- 中间：一条光带连接两者，象征蜕变之旅
- 背景是对标场景：用户与巨匠并肩而立，共同探索知识

**滚动交互：**
- 随着用户滚动，左侧学生逐渐消失
- 右侧领军者逐渐清晰
- 最后定格：用户与巨匠并肩，前方是无限可能

## 五、交互细节设计

### 卡片诞生完整流程

1. **对话阶段**
   - 用户与巨匠Agent自然对话
   - Agent引导用户表达对某个概念的理解

2. **检测阶段**
   - AI检测用户是否"说清楚"了概念
   - 如果没说清，Agent继续追问
   - 如果说清了，进入下一步

3. **夸奖阶段**
   - Agent根据用户实际表现给出针对性夸奖
   - 夸奖示例：
     - "你抓住了本质"
     - "这个举例非常精准"
     - "你已经理解了冯·诺依曼当年的思考"
   - 夸奖文字以动画形式出现

4. **音效阶段**
   - bulingbuling音效播放（清脆、悦耳）
   - 音效时长：0.5秒

5. **卡片诞生**
   - 一张卡片从对话界面中"弹出"
   - 弹出动画：从对话文字中升起，有光效
   - 卡片悬浮在屏幕中央
   - 根据等级，卡片有不同的特效

6. **浏览编辑**
   - 用户可以点击卡片，进入编辑模式
   - 编辑模式：定义、举例、关联、应用四个分栏
   - 用户可以修改任何部分
   - 修改后，AI重新评估等级

7. **归档阶段**
   - 用户确认卡片内容
   - 屏幕右侧出现发光的卡片盒子
   - 用户可以选择：
     - **拖拽归档**：手动拖拽卡片进入盒子
     - **按钮归档**：点击"归档"按钮，卡片变成流光，咻一下钻进盒子

8. **庆祝动画**
   - 如果是SR及以上等级，有简短庆祝动画
   - 如果是UR等级，有全屏庆祝动画

### 音效设计

| 交互 | 音效 |
|------|------|
| **卡片诞生** | bulingbuling音效（清脆、悦耳） |
| **拖拽归档** | 咻音效（快速、流畅） |
| **按钮归档** | 流光音效（持续0.5秒） |
| **获得UR** | 庆祝音效（宏大、激动人心） |

### 卡片盒子设计

**常态（未打开）：**
- 位于屏幕右侧，半透明状态
- 发出柔和的光，象征知识在积累
- 拖拽卡片靠近时，盒子微微张开，发出邀请
- 显示已归档卡片数量

**打开状态（点击盒子）：**

**1. 打开动画**
- 背景逐渐虚化（高斯模糊，从透明度0到0.8）
- 盒子盖子向上掀开（3D翻转动画）
- 卡片从盒子中"流"出，像扑克牌发牌一样
- 卡片横向排列成一条长带，水平居中
- 每张卡片轻微重叠（像扑克牌洗牌后的状态）
- 高等级卡片在前，低等级在后
- 音效：盒子打开音效 + 卡片流动音效

**2. 卡片排列样式**
- 横向滚动，支持鼠标滚轮/触摸滑动
- 卡片间距：相邻卡片重叠30%
- 最左/最右的卡片半透明，提示可以继续滚动
- 鼠标悬停时，卡片微微上浮，阴影加深
- 高等级卡片有流光边框，更加醒目

**3. 卡片查看交互**
- 点击任意卡片：
  - 背景进一步虚化
  - 被点击的卡片放大，移动到屏幕中央
  - 其他卡片淡出或缩小
  - 卡片可以旋转查看（3D效果）
  - 显示卡片详情：定义、举例、关联、应用
- 再次点击/点击关闭：
  - 卡片缩小回到原位
  - 其他卡片恢复
- 拖拽卡片：
  - 可以拖出卡片盒子，放到知识图谱中
  - 建立新的关联

**4. 关闭动画**
- 点击关闭按钮/点击背景
- 卡片快速流回盒子中（逆向动画）
- 盒子盖子关闭
- 背景恢复清晰
- 音效：盒子关闭音效

## 六、技术实现建议

### 前端技术栈

- **框架**：Next.js + React
- **3D渲染**：Three.js / React Three Fiber
- **动画库**：GSAP / Framer Motion
- **滚动交互**：Lenis / ScrollTrigger
- **粒子特效**：Canvas / Three.js

### 插画设计

- **风格**：复古插画 + 3D元素
- **参考**：Shopify Editions Winter2026
- **工具**：Figma / Illustrator
- **3D元素**：Blender / Spline

### 性能优化

- 懒加载插画资源
- 使用WebP格式
- 3D场景按需渲染
- 音效预加载
